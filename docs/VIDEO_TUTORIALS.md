## 🎬 Production Guidelines

### Video Standards
- **Resolution**: 1920x1080 (1080p) minimum
- **Frame Rate**: 30fps or 60fps for smooth terminal animations
- **Audio Quality**: Clear narration, minimal background noise
- **Duration**: Target 3-10 minutes per video for optimal engagement

### Recording Setup
- **Terminal**: Use a clean, consistent terminal theme
- **Font**: Monospace font, 14-16pt size for visibility
- **Window Size**: Consistent terminal dimensions (120x40 recommended)
- **Cursor**: Highlight cursor for visibility during recording

### Content Structure
1. **Introduction** (15-30 seconds): Video topic and learning objectives
2. **Demonstration** (2-7 minutes): Step-by-step walkthrough
3. **Key Points** (30-60 seconds): Summary of important concepts
4. **Next Steps** (15-30 seconds): What to watch next or try next

### Accessibility Features
- **Closed Captions**: Auto-generated with manual review
- **Audio Description**: Describe visual actions for screen reader users
- **Transcript**: Full text transcript for each video
- **Chapter Markers**: Enable jumping to specific sections
